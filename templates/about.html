Welcome to Through Glass Darkly

What is the House?

Through Glass Darkly is an interactive fiction game based on Harry Tuffs' 2D exploration RPG A House of Many Doors , drawing design and aesthetic inspiration from the Failbetter browser game Fallen London .

I, the creator (hello, it's me, Matt), am a huge fan of both games, and I wanted to create an homage of sorts to Harry Tuffs' Herculean labour of love.

You, the player character, are a 'visitant', a being stolen from another world by the parasite dimension that is the 'House'.

It is a mad, disturbing sprawl of wonders and terrors, but you don't have time to worry about all that right now. Plucked from the fantastic dark, only to be pitched through a mirror and set adrift in a kaleidoscopic sea, you are propelled through the House. Blood, quickened by the heart.

Find the next mirror. Delve ever deeper. Do not look back.

Content Warning

Please be aware that this game features themes of death, violence, and body horror, and textual descriptions thereof.

Copyright, Credits and Technical Information

This game was written and designed by Matt Taylor.

It is a work of fan fiction based on Harry Tuffs' A House of Many Doors and is not intended for commercial use. All rights to the original game and its characters, settings, and lore are owned by Harry Tuffs.

This game was created as my final project for Harvard's CS50 course.

The backend is built with Flask, a lightweight Python web framework, and uses a SQLite database to manage user data and story progress.
The frontend uses HTML, CSS, and JavaScript, with Bootstrap providing layout and styling.

You can read much more about the project in the README on the GitHub page .
